비밀번호 안전도

비밀번호에 영문 대소문자, 숫자, 특수문자를 조합하시면 비밀번호 안전도가 높아져 도용의 위험이 줄어듭니다.

  • 영문 대소문자는 구분이 되며, 한가지 문자로만 입력은 위험합니다.
  • 사용가능한 특수문자 : ! " @ # $ % ^ & ' ( ) * + = , - _ . : ; < > ? / ` ~ | { }
  • ID, 주민번호 , 생일, 전화번호 등의 개인정보와 통상 사용 순서대로의 3자 이상 연속 사용은 피해주세요.
  • 비밀번호는 주기적으로 바꾸어 사용하시는 것이 안전합니다.
